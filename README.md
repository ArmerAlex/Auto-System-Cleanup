# Disk Cleanup
#### Instructions on how to delete unnecessary Files to free up Disk space and improve hard drive speed

# Windows
1. On Windows, just press WIN + R to open the run prompt.
2. type "cleanmgr" and press Enter
3. Select the drive partition you want to clean up (normally C:/ for Windows Drive) and press OK
4. After a while, you should see how big the files to be deleted are
5. Check all the checkboxes, except those where you dont want to delete the files and press OK
6. You're done! You should do this again (if you want every week) every month.
